# Mailyan Law - Immigration Attorney Website

A modern, responsive Next.js website for Mailyan Law immigration attorney services.

## Getting Started

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

### Build

```bash
npm run build
npm start
```
